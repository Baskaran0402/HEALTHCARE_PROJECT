# Demo Video

This folder contains demonstration videos of the AI Doctor Assistant application.

## Files

- **app-demo.mp4** - Complete walkthrough of the clinical decision support system

## What the Demo Shows

The demonstration video covers:

1. **Homepage & Authentication**
   - Professional medical UI
   - Healthcare professional verification

2. **Clinical Data Entry**
   - Patient demographics input
   - Vital signs and lab values
   - Medical history checkboxes
   - BMI auto-calculation

3. **AI Analysis**
   - Real-time processing
   - Multi-disease risk assessment
   - ML model predictions

4. **Results & Reports**
   - Risk stratification dashboard
   - Individual disease risk scores
   - Clinical SOAP notes
   - Explainable recommendations

## Usage

The video is referenced in the main README.md file.

### For Presentations

Use this video in:

### For GitHub

When uploading to GitHub:

1. Go to an issue or PR
2. Drag and drop the video
3. Copy the generated URL
4. Update README.md with the URL

## Video Specifications

- **Format**: MP4
- **Duration**: ~2-3 minutes
- **Resolution**: 1080p (recommended)
- **Content**: Full application workflow

---

**Note**: Keep the video under 10MB for GitHub compatibility. If larger, consider:

- Hosting on YouTube (unlisted)
- Using GitHub Releases
- Compressing with HandBrake or similar tools
